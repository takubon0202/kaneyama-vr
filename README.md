# ヤマザワ 金山中央公民館デジタル店 — VRシェルフ UI プロトタイプ

縦長 16:9（1080×1920・実機75インチ想定）のデジタルサイネージ / VRシェルフ UI プロトタイプです。

🔗 **公開URL:** https://takubon0202.github.io/kaneyama-vr/

## 特徴

- **店内マップ**：一般的なスーパーの順路（黄金ルート）に基づく反時計回りの周回動線
  `入口 → ①青果 → ②鮮魚 → ③精肉 → ④日配 → ⑤冷凍 → グロサリー → ⑥惣菜・パン → レジ`
- **売場タッチ → 商品一覧 → 商品詳細 → カート** の操作フロー
- **本日のおすすめ・特売**：全売場の割引商品・イチオシを集約（%OFF・取消線価格表示）
- **商品画像**：全102点（白背景の商品写真）
- **ヤマザワ ブランド**：コーポレートカラー（青・緑・白）に準拠

## 構成

- `index.html` … 単一ファイル完結（HTML/CSS/JS、外部依存なし）
- `images/` … 商品画像（`<売場id>_<番号>.png`）
- `gen_images.py` … 商品画像の生成スクリプト（Codex CLI 組み込み image_gen を使用）
